This course covers topics beyond the basics of using git and GitHub that is covered in the introductory course.

Link to the video

In this class, participants will:

  • get a brief exposure to the git command line interface (CLI), and how to link it to what is happening in a git GUI
  • learn a simple single-user workflow for small projects
  • learn a simple multi-user workflow for small and medium-sized projects
  • learn how to anticipate, avoid and deal with merge conflicts
  • learn how to use git functionalities like stashing, resetting and cherry-picking
  • learn about software project best practices: readme, contribution guide, licences, conda environment files

The prerequisites of this class are the things covered in the introductory tutorial:

  • have a GitHub account and a GitKraken installation
  • be familiar with the basic navigation of GitHub repositories and the GitKraken GUI
  • know how to clone a repo, create, switch and delete branches in GitKraken, and how to open a pull request on GitHub

Since we are diving into more advanced git topics here, we will be providing one single example that runs a simple Python script. No previous Python knowledge is required. The rest will be taught, as previously, exclusively on text files to avoid coding distructions (and possible bugs...)

The technical requirements for this class are:

  • a GitHub account (more details below)
  • an installation of GitKraken (more details below) - this GUI is available for all platforms, the free version is sufficient for this tutorial
  • optional: a conda installation - this can be Miniconda or Anaconda (see Miniconda installation instructions below) - this will allow you to follow the one (short but very useful) Python example but this will be extremely short and equally instructional if you just watch

Installation requirements for Advanced Git for Astronomers:

  • Create a GitHub account
    • 1. go to https://github.com
    • 2. click on the link to "Sign up"
    • 3. provide a username, email address and password and create your account
    • 4. validate the confirmation email
  • Install GitKraken
    • 1. go to https://www.gitkraken.com/download
    • 2. download the appropriate installer for your operating system
    • 3. run the installation
    • 4. when you open the app for the first time it will prompt you to sign in - use your GitHub account for this, and the email you used to create your GitHub account
  • Install Conda and git CLI (optional!)
    • On Windows, you can use git CLI and conda easily through "Git Bash":
      • 1. go to the Miniconda website and install miniconda for Python 3.7 for your respective OS: https://docs.conda.io/en/latest/miniconda.html
      • 2. make sure to add Miniconda to your PATH during the installation, even if it is marked as not recommended. This is required so that Git Bash can recognise it (see next point).
      • 3. run the installation
      • 4. install Git Bash from here: https://gitforwindows.org/
    • On MacOS, follow point 1. above to get Miniconda. To install the git CLI, simply run "git --version" in your terminal - either you already have it installed, or it will prompt you with instructions.
    • On Linux, follow point 1. above to get Miniconda, and see here how to install the git CLI: https://git-scm.com/download/linux
